# Test 3 - Function Overloading

This test verifies function overloading based on different parameter signature types in GigglyCode.

## Verified Features:
- **Int Overload:** `def subtract(a: int, b: int) -> int`
- **Float Overload:** `def subtract(a: float, b: float) -> float`
- **Resolution:** The compiler resolves which overload to call based on the argument types (integers vs floating-point literals).
