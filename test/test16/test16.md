# Test 16 - Enum and String Switch Case Statements

This test verifies that the `switch` statement supports enum constants and string constants in GigglyCode.

## Verified Features:
- **Enum Switches:** Switching on an enum instance and matching against enum constants.
- **String Switches:** Switching on a string and matching against string literals (strcmp-based cascade).
- **Fallback Case:** The `other` block, which is executed if no matching case is found.
