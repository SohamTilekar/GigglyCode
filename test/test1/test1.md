# Test 1 - Structs & Struct Methods

This test verifies the definition and usage of structs and member methods in GigglyCode.

## Verified Features:
- **Struct Declaration:** `struct Rectangle { ... }` with member fields `width` and `height`.
- **Constructor:** Custom initializer method `def __init__(self: Rectangle, width: int, height: int)` which sets the fields.
- **Member Methods:** `def area(self: Rectangle) -> int` and `def perimeter(self: Rectangle) -> int`.
- **Instance Creation & Method Calls:** Creating a `Rectangle` instance and calling its methods.
