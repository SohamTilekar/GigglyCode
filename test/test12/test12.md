# Test 12 - Module Imports

This test verifies the `import` mechanism in GigglyCode.

## Verified Features:
- **Module Import:** `import "modules/math_utils";` which parses and compiles `modules/math_utils.gc` in the project.
- **Namespaced Member Call:** Accessing functions in the imported module using namespaced notation `math_utils.add(10, 20)`.
