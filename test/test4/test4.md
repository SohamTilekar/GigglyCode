# Test 4 - Switch Case Statement

This test verifies the `switch` statement flow control structure in GigglyCode.

## Verified Features:
- **Switch Case Block:** `switch (val) { case (1) { ... } case (2) { ... } ... }`
- **Fallback Case:** The `other` block, which is executed if no matching case is found.
- **Literal Cases:** Constant case evaluation.
