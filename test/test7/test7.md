# Test 7 - Loop Modifiers (NotBreak & IfBreak)

This test verifies the loop continuation and termination blocks: `notbreak` and `ifbreak`.

## Verified Features:
- **`notbreak` block:** Runs if the loop completes execution naturally (when the loop condition evaluates to false).
- **`ifbreak` block:** Runs if the loop is aborted early using the `break` statement.
