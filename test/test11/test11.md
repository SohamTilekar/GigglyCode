# Test 11 - Enums

This test verifies enum declarations and usages in GigglyCode.

## Verified Features:
- **Enum Declaration:** `enum Color { RED; GREEN; BLUE; }` where each variant is separated by a semicolon.
- **Enum Field Access:** Retrieving constant values using `Color.GREEN`.
- **String Conversion Method:** Using the built-in `.getName()` method on enum instances to fetch the string literal of the enum value.
