# Test 5 - Raw Arrays

This test verifies raw arrays in GigglyCode.

## Verified Features:
- **Raw Array Allocation:** `arr: raw_array[int] = raw_array(int, 5);` allocating 5 integers in contiguous memory.
- **Array Indexing:** Subscript operator access: reading and writing elements like `arr[i] = i * 10;`.
